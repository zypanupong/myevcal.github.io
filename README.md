# คำนวณค่าไฟชาร์จรถ EV ที่บ้าน

เว็บแอปหน้าเดียว (single-file HTML) สำหรับคำนวณค่าไฟฟ้าจากการชาร์จรถยนต์ไฟฟ้าที่บ้าน ปรับแต่งไว้สำหรับ **AION V 500 Premium** (แบตเตอรี่ 64.5 kWh, รับไฟ AC สูงสุด 6.6 kW) แต่ปรับค่าให้เข้ากับรถรุ่นอื่นได้จากหน้าตั้งค่า

ทดลองใช้งานได้ทันทีที่: `index.html` (เปิดในเบราว์เซอร์ได้เลย ไม่ต้องติดตั้งอะไรเพิ่ม)

## ฟีเจอร์

- กรอกข้อมูลการชาร์จได้ 3 แบบ: หน่วยไฟ (kWh) ตรงๆ, % แบตเตอรี่ก่อน–หลังชาร์จ (คิด loss ให้อัตโนมัติ), หรือกำลังชาร์จ (kW) × เวลา (ชั่วโมง)
- คำนวณค่าไฟจากอัตราขั้นสูงสุดของการไฟฟ้า (ประเภทบ้านอยู่อาศัย 1.2) รวมค่า Ft และ VAT ให้อัตโนมัติ — ปรับตัวเลขอัตราได้เองเมื่อมีประกาศอัตราใหม่
- เลือกวันที่ชาร์จ พร้อมประมาณเวลาชาร์จจากกำลังไฟของเครื่องชาร์จ
- สรุปรายการชาร์จทั้งแบบรายครั้งและรายเดือน พร้อมกราฟแท่งเปรียบเทียบค่าไฟ/หน่วยไฟ
- ดาวน์โหลดประวัติการชาร์จเป็นไฟล์ CSV
- **จดจำข้อมูลอัตโนมัติ**: ประวัติการชาร์จและค่าตั้งอัตราไฟถูกบันทึกไว้ในเบราว์เซอร์ (localStorage) ให้อัตโนมัติ ปิดแอปหรือรีเฟรชหน้าแล้วข้อมูลไม่หาย (ยังอยู่เฉพาะเบราว์เซอร์/เครื่องเดิมที่ใช้งาน)
- **สำรอง/กู้คืนขึ้น Google Drive (ไม่บังคับ)**: เชื่อมต่อบัญชี Google ของคุณเองเพื่อสำรองข้อมูลขึ้น Drive และดึงกลับมาใช้บนเครื่องหรือเบราว์เซอร์อื่นได้ (ดูวิธีตั้งค่าด้านล่าง)

## วิธีใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ใดก็ได้ (Chrome, Edge, Safari) หรือดูออนไลน์ผ่าน GitHub Pages หากเปิดใช้งานไว้ (ดูหัวข้อด้านล่าง)

## เปิดใช้งานผ่าน GitHub Pages (ไม่บังคับ)

1. ไปที่ Settings → Pages ของ repository นี้
2. เลือก Source เป็น branch `main` โฟลเดอร์ `/ (root)`
3. บันทึก แล้วรอสักครู่ จะได้ลิงก์ให้เปิดใช้งานผ่านเว็บ

## ติดตั้งเป็นแอปบน Android (PWA)

โปรเจกต์นี้เป็น Progressive Web App (PWA) เต็มรูปแบบ — มี `manifest.json`, `service-worker.js` และไอคอนแอปพร้อมแล้ว วิธีติดตั้งลงเครื่อง Android:

1. เปิดลิงก์ GitHub Pages ของโปรเจกต์นี้ด้วย **Chrome บน Android** (ต้องเป็น https:// — service worker และปุ่มติดตั้งจะไม่ทำงานถ้าเปิดจากไฟล์ในเครื่องโดยตรง)
2. แตะเมนู ⋮ มุมขวาบน แล้วเลือก **"ติดตั้งแอป"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
3. ยืนยัน จะได้ไอคอนแอปบนหน้าจอหลัก เปิดแล้วแสดงผลเต็มจอเหมือนแอปทั่วไป ไม่มีแถบ URL ของเบราว์เซอร์ และใช้งานได้แม้ไม่มีอินเทอร์เน็ต (service worker แคชไฟล์ไว้ให้)

### อยากได้ไฟล์ .apk จริงๆ

Progressive Web App ที่ตั้งค่าไว้นี้สามารถแปลงเป็นไฟล์ .apk ที่ติดตั้งได้จริงโดยไม่ต้องเขียนโค้ดเพิ่ม ผ่านบริการฟรีของ Google:

1. เปิดใช้งาน GitHub Pages ให้เรียบร้อยก่อน (จะได้ URL แบบ `https://<username>.github.io/<repo>/`)
2. ไปที่ [pwabuilder.com](https://www.pwabuilder.com) แล้ววาง URL ของเว็บแอป
3. เลือกแท็บ **Android** แล้วกด **Generate Package** จะได้ไฟล์ .apk / .aab ที่เซ็นชื่อพร้อมติดตั้งหรือส่งขึ้น Play Store ได้ทันที

(หมายเหตุ: การ build APK แบบ native ในเครื่อง requires Android SDK ที่มีขนาดหลาย GB — pwabuilder.com ทำขั้นตอนนี้ให้บนคลาวด์ ไม่ต้องติดตั้งอะไรเพิ่มที่เครื่อง)

## สำรองข้อมูลขึ้น Google Drive (ไม่บังคับ)

แอปจดจำข้อมูลในเบราว์เซอร์ให้อัตโนมัติอยู่แล้ว ส่วนนี้ใช้เฉพาะกรณีอยากสำรองข้อมูล หรือย้าย/ใช้ข้อมูลเดียวกันข้ามเครื่อง/เบราว์เซอร์ ผ่านบัญชี Google Drive ของคุณเอง

เนื่องจากการเชื่อมต่อ Google Drive ต้องผูกกับบัญชี Google Cloud ของผู้ใช้แต่ละคน (Claude ไม่สามารถสร้าง Client ID แทนให้ได้) จึงต้องตั้งค่าเองครั้งเดียวตามขั้นตอนนี้:

1. เปิด [Google Cloud Console](https://console.cloud.google.com/) แล้วสร้างโปรเจกต์ใหม่ (หรือใช้โปรเจกต์เดิมก็ได้)
2. ไปที่ **APIs & Services → Library** ค้นหา "Google Drive API" แล้วกด **Enable**
3. ไปที่ **APIs & Services → OAuth consent screen**
   - เลือก User Type เป็น **External**
   - กรอกชื่อแอป (เช่น "EV Charge Calculator"), อีเมลติดต่อ แล้วบันทึก
   - ในหน้า **Test users** ให้เพิ่มอีเมล Gmail ของคุณเองเป็น test user (ไม่ต้องส่งขอ verify จาก Google เพราะใช้เองคนเดียว แอปจะอยู่ในสถานะ "Testing" ตลอดได้ ใช้งานได้ปกติ)
4. ไปที่ **APIs & Services → Credentials → Create Credentials → OAuth client ID**
   - Application type เลือก **Web application**
   - ในช่อง **Authorized JavaScript origins** ใส่ URL ของเว็บแอปนี้ เช่น `https://<username>.github.io` (ใส่เฉพาะโดเมน ไม่ต้องมี path ต่อท้าย)
   - ไม่ต้องกรอก Authorized redirect URIs
   - กด Create จะได้ **Client ID** ที่ลงท้ายด้วย `.apps.googleusercontent.com`
5. คัดลอก Client ID มาวางในแอป ที่การ์ด "☁️ สำรองข้อมูลขึ้น Google Drive" → เปิด "⚙️ ตั้งค่าการเชื่อมต่อ Google Drive" → วางในช่อง Client ID
6. กด **"🔗 เชื่อมต่อ Google Drive"** แล้วเข้าสู่ระบบ/อนุญาตสิทธิ์ด้วยบัญชี Google ที่เพิ่มเป็น test user ไว้ในขั้นตอนที่ 3
7. หลังเชื่อมต่อสำเร็จ ใช้ปุ่ม **"☁️⬆ บันทึกขึ้น Drive"** เพื่อสำรองข้อมูลปัจจุบัน หรือ **"☁️⬇ โหลดจาก Drive"** เพื่อดึงข้อมูลที่เคยสำรองไว้กลับมา (จะมีการถามยืนยันก่อนเขียนทับข้อมูลในเครื่อง)
8. ถ้าต้องการให้ซิงค์ขึ้น Drive อัตโนมัติทุกครั้งที่บันทึก/ลบรายการ ให้ติ๊กช่อง "ซิงค์ขึ้น Drive อัตโนมัติ" หลังเชื่อมต่อแล้ว

**หมายเหตุด้านความปลอดภัย**: แอปขอสิทธิ์แบบจำกัด (`drive.file` scope) จึงเข้าถึงได้เฉพาะไฟล์เดียวที่แอปสร้างขึ้นเอง (`ev_charge_data.json`) เท่านั้น ไม่สามารถเห็นหรือแก้ไขไฟล์อื่นใน Google Drive ของคุณได้ และ Client ID ที่กรอกไว้จะถูกเก็บไว้ในเบราว์เซอร์ของคุณเท่านั้น ไม่ถูกส่งไปที่อื่น การเชื่อมต่อ (access token) จะหมดอายุตามปกติของ Google และต้องกดเชื่อมต่อใหม่ทุกครั้งที่เปิดแอปใหม่ (เพื่อความปลอดภัย ไม่มีการเก็บ token ถาวรไว้ในเครื่อง)

## หมายเหตุ

- อัตราค่าไฟที่ตั้งไว้เริ่มต้นอ้างอิงงวด พ.ค.–ส.ค. 2569 (ค่า Ft = 0.1623 บาท/หน่วย) แก้ไขได้จากส่วน "ตั้งค่าอัตราค่าไฟ" ในหน้าเว็บเมื่อมีประกาศอัตราใหม่
- ภาพพื้นหลังในหน้าเว็บเป็นภาพถ่ายส่วนตัว ก่อนเผยแพร่ต่อสาธารณะ (repo แบบ public) ควรพิจารณาว่าต้องการให้เห็นป้ายทะเบียนรถในภาพหรือไม่
- ข้อมูลที่จดจำอัตโนมัติ (localStorage) และไฟล์สำรองใน Google Drive อยู่ในอุปกรณ์/บัญชีของคุณเองทั้งหมด ไม่มีการส่งข้อมูลไปเซิร์ฟเวอร์อื่นใดนอกจาก Google Drive ของคุณ (เฉพาะกรณีเปิดใช้ฟีเจอร์สำรองขึ้น Drive)
